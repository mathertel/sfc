Form Data and Form Elements Test

Dialog Form-Data in JSON

---

This is data updated by fetching the JSON data Object from the form using the getJsonData() method of the <u-form-json> component.

What to test here...

This page tests form data handling with various input elements:

  • text input
  • checkbox
  • duration input (u-duration)
  • switch (u-switch)

Interact with the form, submit to view JSON data, reset, or use buttons to toggle the switch. JSON data updates live every 500ms.
